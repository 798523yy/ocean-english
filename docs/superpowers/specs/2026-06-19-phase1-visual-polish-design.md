# Phase 1: 水族馆视觉生动化 (Visual Polish)

**日期**: 2026-06-19  
**状态**: 已确认  
**范围**: 纯视觉升级，无需外部资源

---

## 目标

在 2D Canvas 基础上大幅提升水族馆的视觉品质，打开网页即肉眼可见变化。不改架构、不加依赖。

## 四项改造

### 1. 粒子特效增强 (`particles.js`)

| 改动 | 说明 |
|------|------|
| 气泡系统升级 | 25→40个最大气泡，沙地随机生成，上升带水平摇摆，水面爆裂波纹 |
| 水下光柱 (God Rays) | 4条光线从水面射入，随波动偏移，浮游生物在光柱内更亮 |
| 环境微粒 | 珊瑚/海葵旁散发细微闪光点，legendary生物带光晕环 |
| 喂食粒子 | 点击喂食时更多食物颗粒（12→20），更散开 |

### 2. 水面/水底特效 (`aquarium.js`, `lighting.js`)

| 改动 | 说明 |
|------|------|
| 动态水面线 | 画布顶部绘制正弦波叠加水面，半透明蓝绿渐变 |
| 焦散光升级 | 沙地光斑密度翻倍，随深度衰减，移动更流畅 |
| 沙地动画 | 沙纹慢速波动，沙地随机冒气泡 |

### 3. 生物动画增强 (`swim-ai.js`)

| 改动 | 说明 |
|------|------|
| 缓入缓出 | `easeInOut` 加速度替代恒速游动 |
| 转弯弧线 | 方向变化时走贝塞尔弧线而非瞬切 |
| 浮动效果 | 停留时有上下微浮动 |
| 冲刺行为 | 偶尔快速冲刺短距离 |
| 聚群 | 同种生物有弱吸引力 |
| 体型差异 | 同种生物 ±15% 大小随机 |
| 尾迹粒子 | 游动快时尾迹更明显 |
| 传说光晕 | legendary 生物有柔和外发光 |

### 4. 色彩主题深化 (`lighting.js`, `main.css`)

| 改动 | 说明 |
|------|------|
| 时段色调 | Dawn 暖橙→蓝 / Day 清澈蓝绿 / Dusk 紫粉→蓝 / Night 深蓝+荧光 |
| 渐变过渡 | 时段切换 0.5s → 3s 缓慢过渡 |
| 夜晚荧光 | Night 模式生物自发光 (bioluminescence) |
| 沙地色调 | 沙地随时间段变色 |

## 技术约束

- **无新依赖**：不改动 requirements.txt
- **不新增文件**：全部在现有 JS/CSS 模块上增强
- **不改架构**：保留 Canvas 2D 渲染管线
- **性能要求**：移动端保持 ≥30fps，桌面端 ≥50fps

## 改动范围

| 文件 | 改动量 | 说明 |
|------|--------|------|
| `particles.js` | ~150行 | 气泡系统重写、光柱粒子、环境微粒 |
| `lighting.js` | ~100行 | 焦散升级、时段过渡、生物荧光 |
| `environment.js` | ~50行 | 沙地动画、珊瑚微粒 |
| `swim-ai.js` | ~120行 | 缓入缓出、弧线转弯、冲刺、聚群 |
| `aquarium.js` | ~80行 | 水面线绘制、生物大小变异、尾迹 |
| `main.css` | ~40行 | 时段渐变速度、色彩变量调整 |
