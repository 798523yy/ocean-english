📋 项目概述

一款以海洋水族馆为主题的英语学习 Web 应用，通过游戏化机制激励用户每日练习英语。

🔧 技术选型理由

为什么不用框架？

  • 项目规模适中，原生 JS 完全够用
  • 零依赖，加载速度快
  • 适合学习型项目，逻辑清晰
  • 不引入 React/Vue 的学习成本

为什么选 Flask？

  • Python 开发效率高
  • 轻量级，适合小项目快速迭代
  • JSON 文件存储，无需数据库
  • PythonAnywhere 一键部署

为什么用 Canvas？

  • 水族馆需要动态粒子效果
  • Canvas 性能优于 DOM 动画
  • 每个生物可独立运动轨迹
  • 支持未来扩展复杂动画
